Various tools used by cybersecurity analyst :
1. SIEM (security Information and Event Management) : here the logs of critical activities conducted in a organization are analyzed, a analyzer has a lot of logs and data to analyze so SIEM tool is used which only tells about the logs which are identified as certain threat, it creates dashboard, sorts logs
   There are two types of tools used in SIEM tool - 1) splunk 2) Chronicle
2. Security Protocol analyzer : It observes the network traffic of the entire organization alerts about login, log out of computers which are connected to network
3. Playbooks : it is like a mannual which a organization provides, it varies from organization to organization depending on the tasks it has all the information about how a incident, breach of the data can be handeled
   it is of two types 
   1. Chain of custudy
   2. protecting and preserving of evidencies
