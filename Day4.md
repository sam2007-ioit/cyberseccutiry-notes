Various Concepts of Cybersecurity :
  1. Programming : mainly two types of programming languages are used by cybersecurity professionals they are-
       A. Python - it is used to give set of instruction to the computer that has to be executed in certain situations
       B. SQL (structured query language) - here we interact, request and pull the data from database this also use to filter the data only taking the data which is needed at the time
  2. Operating system - the interface between user and computer hardware there are multiple operating systems like windows, macOS, linux etc. 
                        linux is a open source operating system, means you can contribute to make the system better
  3. Web vulnerability - a unique point in the web application which is exploited by the threat actors in order to steal data, making changes to web applications
  4. Antivirus application - a application that detects and removes malware from the computer it is also called as antimalware
  5. Intrusion detection system (IDS) - a system which keep an eye on all the activities going on the device and alert if any suspecious activity found
  6. Encryption - it is the process of converting the readable data into cryptographically encoded language so no unauthorized person can read it, in cryptography plaintext is converted into chipertext
  7. Penetration test - a test were a stimulation is created of cyber attack to check the vulnerablility and weakness
